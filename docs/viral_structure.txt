# Language Plugins

Each language gets a plugin.

Example:

```
plugins/languages/python
plugins/languages/typescript
plugins/languages/go
plugins/languages/rust
```

Each defines:

```
AST queries
dependency rules
signature extraction
```

Example:

```python
class PythonPlugin(LanguagePlugin):
    def extract_functions(self, ast):
        ...
```
